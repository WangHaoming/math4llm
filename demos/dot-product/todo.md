### 4.2 决策边界:一条与 w 垂直的线

神经元打的分有正有负,正负的分界线在哪?在 **w · x + b = 0** 的地方。
在二维里这是一条直线,在高维里是一张"超平面",而 **w 恰好垂直于它**:

```
                ^  w   (神经元关心的方向, 垂直于边界)
                |
                |      w·x + b > 0    兴奋区: x 的影子够长, 打分为正
                |
================+================    决策边界: w·x + b = 0
                |
                       w·x + b < 0    沉默区: 影子不够, 打分为负
```

这张图值得多看一会儿,它是"神经网络如何思考"的最小单元:

- **w 定方向**:边界一定垂直于 w。想让边界转个角度?转 w 就行;
- **b 定位置**:b 把边界沿 w 方向平移。b 越大,边界向"沉默区"退,
  兴奋区越大——它调的是"多像才算像"的门槛;
- **打分即距离**:w·x + b 的大小(除以 |w|)就是 x 到边界的**有向距离**。
  分数越正,x 在兴奋区扎得越深。

> **一个神经元 = 在输入空间里立一张与 w 垂直的隔板,把空间切成两半,
> 然后报告每个输入落在哪一侧、离隔板多远。**

顺带就把 ReLU 也看懂了:ReLU 把负分数压成 0,几何上就是——
**隔板背面的一切,一律当作"影子为零"处理**。神经元从此只对自己那半边空间
"睁眼",成为一个真正有选择性的探测器("没有这个特征"和"有一个反特征"
从此是两回事)。这也是为什么在 what-is-nn 演示里关掉 ReLU 后,
两层网络会塌缩成一层:没有隔板的截断,一串点积连乘还是一个点积,
再多层也只能画一条直的边界。
