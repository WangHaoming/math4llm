# 向量加法 · 点积

> 拖动两个向量,直观理解「加法」与「点积」到底在做什么——并从 cos 的几何意义
> 一路讲到:点积就是神经网络里最重要的那一次乘法。

## 如何使用

1. 打开本目录的 `index.html`(或从合集首页进入)。
2. 画布上有两个箭头:**a**(红) 和 **b**(蓝)。
3. **拖动箭头端点**改变向量;右侧面板实时显示坐标、点积、夹角、投影。
4. 三个开关:
  - **显示平行四边形 a + b**:看向量加法的几何构造(绿色)。
  - **显示投影**:把 a 投影到 b 上(黄色箭头),这就是点积的几何意义。白色虚线是 a 端点到 b 的「垂线」,垂足就是投影的终点。
  - **证明演示:把 b 转到 x 轴上**:整个图形刚性旋转(长度、夹角都不变),
    b 躺平到 x 轴后,画面上标出 a 的横竖分解(影子 |a|cosθ、举高 |a|sinθ)
    和夹角 θ,面板里列出乘加的每一步数值——这是 3.3 那个证明的动画版。

下面的讲解建议一边读、一边拖着箭头对照着玩——文中所有"试试"都指这个演示。

## 〇、先说结论

整篇讲解要说清楚的,其实是一句话:

> **点积 = 「方向一致度」× 两个长度。而神经网络的每一次"识别",本质上都是
> 拿输入和存好的模式做一次点积——问一句:"你在我关心的方向上,影子有多长?"**

这句话现在看可能没什么感觉。我们先热个身,然后从最底层的那个零件——cos——
开始,一步一步把它拆开。

## 一、热身:向量加法 = 首尾相接

向量 `a + b` 的意思是:**先沿 a 走,再沿 b 走**,最终到达的位置。

把 b 平移到 a 的末端(或反过来),从原点指向终点的那个箭头就是 `a + b`。
两条平移得到的虚线和原来的 a、b 围成一个**平行四边形**,`a + b` 正是它的
对角线——这就是"平行四边形法则"。打开演示的第一个开关就能看到这个构造。

> 物理直觉:两个力同时作用在一个物体上,合力就是它们的向量和;
>代入神经网络中应用，当 a 表示一个特征向量时，a + b 相当于给特征 a 朝着 b 的方向上做一个偏移。

加法讲完了。它回答的是"两个向量**合起来**是什么";而接下来的点积回答的是
另一个更深的问题:"两个向量**有多一致**"。 

> 注： 向量相加暂时在最初级的神经网络中没有得到应用，但是在 Transformer 中，会在很多地方被使用，所以在这里我把它作为讲解点积前的热身内容讲解一下，大家也不要忽视它，在之后我们学习 GPT 模型的时候，它还是相当重要的。接下来让我们进入正题，向量的点积。

## 二、cos 在量什么:一把"方向一致度"的尺子

### 2.1 先忘掉三角形,看影子

讲向量点积之前，我要想讲一讲 cos，它对理解向量点积非常重要。

我们通常对 cos 的记忆停留在"直角三角形里邻边比斜边"。这个定义没错,
如果用更形象的方式来解释，cos 真正在量的东西,用一个影子就能看清。

想象一根长度为 1 的杆子,一端钉在原点 O,另一端 P 可以绕着转。
太阳从头顶垂直照下来,杆子在地面(x 轴)上投出一道影子:

```
        y
        |
        |      * P    (杆子另一端, 杆长 OP = 1)
        |     /.
        |    / .
        |   /  .    <- 光线垂直落下, 这段竖直虚线的高度 = sin θ
        |  /   .
        | / θ  .
--------*------+-------------- x (地面)
        O <---->
         影子长度 = cos θ

        杆子与地面的夹角是 θ,影子的长度就是 cos θ
```

杆长固定是 1,所以影子多长,完全由**方向**决定:

- 杆子**平躺**在地上(θ = 0°):影子和杆子一样长,cos θ = 1;
- 杆子**斜着**(0° < θ < 90°):影子缩短,0 < cos θ < 1;
- 杆子**竖直立起**(θ = 90°):影子缩成一个点,cos θ = 0;
- 杆子**倒向另一边**(θ > 90°):影子落在 x 轴的**负方向**,cos θ < 0;
- 杆子**完全反向平躺**(θ = 180°):cos θ = −1。

把这五种情形排在一条线上,你会得到一把尺子:

```
cos θ:   +1 <------------------ 0 ------------------> -1
```

| | 同向 | 垂直 | 反向 |
| --- | --- | --- | --- |
| θ | 0° | 90° | 180° |
| cos θ | **+1** | **0** | **−1** |
| 一句话 | 完全一致 | 互不沾边 | 完全相反 |

> **cos θ 是一把"方向一致度"的尺子:+1 分表示方向完全一致,0 分表示互相垂直、
> 谁也帮不上谁,−1 分表示完全唱反调。它只看方向,不看长短。**

"只看方向,不看长短"这一点值得停下来体会:杆子是 1 米还是 100 米,
θ 相同则 cos θ 相同。cos 把"长度"这件事完全剥离了,剩下的是纯粹的方向关系。
这正是它能当"打分标准"的原因——评分标准本身不应该随尺度变化。

**顺带把 sin θ 也认了**(第三节的证明马上要用):图里那段竖直虚线——
P 离地面的**高度**——就是 sin θ。cos 和 sin 是同一根杆子的一横一竖两个成分:

$$\text{cos θ 记录"贴地多少",sin θ 记录"离地多少"}$$

一躺一立,合起来正好还原整根杆子——勾股定理说的就是这件事:
$\cos^2\theta + \sin^2\theta = 1$(杆长的平方)。于是单位杆子端点 P 的坐标
就是 $(\cos\theta,\ \sin\theta)$;把杆子放大到长度 $|\mathbf{a}|$,端点坐标
就是 $(|\mathbf{a}|\cos\theta,\ |\mathbf{a}|\sin\theta)$。**记住这个
"横竖分解"**,第三节的证明用它一步就能算完。

### 2.2 旁注:为什么垂直恰好是 0 分?

尺子上最有内容的刻度其实是中间那个 0。θ = 90° 时影子缩成一个点,
意味着:**无论沿 P 方向走多远,在 x 方向上都寸步未行**。两个方向互相垂直,
就是"你的努力对我毫无贡献"——不帮忙(正)也不拆台(负),干脆利落的零。
这个"零贡献 = 垂直"的直觉,到了神经网络那节会变成"神经元对这个输入毫无反应"。

## 三、点积 = 影子 × 长度

### 3.1 计算规则:乘一乘,加一加

先说怎么算。点积的规则非常简单:假设有两个向量：

$$[a_x, a_y],[b_x, b_y] $$

,计算这两个向量的点积,就是把相同位置的值相乘然后相加（记住这是计算规则，你不需要想为什么要这么算，后面我们会解释这样计算的意义）:

$$\mathbf{a} \cdot \mathbf{b} = a_x b_x + a_y b_y$$

比如 $\mathbf{a} = (2,\, 1)$,$\mathbf{b} = (3,\, 4)$:

$$\mathbf{a} \cdot \mathbf{b} = 2 \times 3 + 1 \times 4 = 10$$

维度升高,规则不变——$n$ 维就是 $n$ 对"相乘再相加":

$$\mathbf{a} \cdot \mathbf{b} = \sum_{i=1}^{n} a_i b_i$$

全程只有乘法和加法,连量角器都不用——演示面板里显示的点积,`sketch.js`源码中
（感兴趣的读者可以自己去读一读源码）就是老老实实算 `a.x*b.x + a.y*b.y`。

规则会算了,但两个问题立刻冒出来:**这个数是什么意思?为什么偏偏是
"对应相乘再相加"?** 答案在几何里。

### 3.2 几何意义:影子 × 长度

上一节的杆子长度固定为 1。现在把它换成任意向量 **a**(长度 |a|,随便多长),
影子照样投,只是等比例放大:

$$\text{a 在某方向上的影子长度} = |\mathbf{a}| \cos\theta$$

这个量叫 **a 的投影**(准确说是标量投影)。它继承了 cos 的符号:
θ 超过 90° 时影子落到反方向,投影为负。

现在,地面方向也换成一个向量 **b**。要给出的说法是:**3.1 那串"乘加",
算出来的恰好是"影子 × 长度"**:

```
            a
           *
          /.
         / .
        /  .     <- 从 a 的端点向 b 所在直线作垂线
       /   .
      / θ  .
     *-----+----------->  b
     O <--->
        p = |a|cosθ   (a 投在 b 方向上的影子)

     a · b = p × |b| = |a||b|cosθ   (影子长度 × b 的长度)
```

> **点积 $\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}||\mathbf{b}|\cos\theta$:先用 cos 给"方向一致度"打分,
> 再乘上两个向量各自的长度。它是"方向分"与"力度"的乘积。**

打开演示,勾选"显示投影",黄色箭头就是 a 投到 b 上的影子;面板里
「a 在 b 上的投影」这一行,算的就是 (a·b)/|b|。试试这几种情形:

- 把 a 转到和 b **同向**:θ→0°,影子最长,点积最大;
- 转到**垂直**:黄色箭头缩成一个点,点积 = 0;
- 转过 90° 继续转:影子探向 b 的**反方向**,点积变负;
- 方向都不动,**只把 b 拉长**:夹角没变、影子没变,点积却成比例变大——
  因为"力度"那一项变了。

### 3.3 为什么"乘加"恰好等于"影子 × 长度"?

3.2 到目前为止只是个"声明",还没证。证明比想象中简单——不需要任何
现成的定理,只需要想起一件事:**"地面"(x 轴)本来就是我们随手画的**。
向量的长度、它们之间的夹角、影子的长短,都是几何事实,不依赖坐标轴
画在哪。那就干脆把 x 轴转到和 b 重合,让 b 舒舒服服躺在地上:

```
        y     a
        |     *
        |    /.
        |   / . 
        |  /  .   <- |a|sinθ  a 里"举离地面"的成分
        | / θ . 
--------*-----+----------*---> x
        O <-->          b = (|b|, 0)
         |a|cosθ  a 里"贴着地面"的成分(正是影子)
```

在这副坐标里,两个向量的坐标都能直接写出来:

$$\mathbf{b} = (|\mathbf{b}|,\ 0) \qquad \text{(躺在 x 轴上,纵坐标为 0)}$$

$$\mathbf{a} = (|\mathbf{a}|\cos\theta,\ |\mathbf{a}|\sin\theta) \qquad \text{(横坐标是影子,纵坐标是举高的部分)}$$

代入 3.1 的计算规则,一行算完:

$$\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}|\cos\theta \cdot |\mathbf{b}| + |\mathbf{a}|\sin\theta \cdot 0 = |\mathbf{a}|\,|\mathbf{b}|\cos\theta$$

证完了。注意 sinθ 那一项的下场:a 里**垂直于 b 的成分**,乘上的是 b 的
纵坐标 0,被干脆利落地丢掉了。这正是"对应相乘再相加"这个规则的用意所在:

> **点积只认影子:a 里平行于 b 的成分被 |b| 放大计入,垂直于 b 的成分
> 一律无视。**2.2 里"垂直 = 零贡献"的直觉,在公式里就体现为 sinθ 项乘零。

这个证明在演示里有动画版:勾选「**证明演示:把 b 转到 x 轴上**」,随手拖出
两个向量,整个图形就会保持夹角不变地旋转,b 缓缓躺平;影子(黄)、举高
(白虚线)、θ 圆弧和乘加的每一步数值都标在图上。顺便还能验证一件事:
转轴前后,面板里的点积 a·b **一个数都没变**——这正是下面旁注要说的。

**旁注:转坐标轴,"乘加"的结果不会变吗?** 有此一问,说明你想到了关键处。
把任意坐标下的 a、b 都写成"长度 ×(cos方位角, sin方位角)"的形式: $\mathbf{a} = |\mathbf{a}|(\cos\alpha,\ \sin\alpha)$,$\mathbf{b} = |\mathbf{b}|(\cos\beta,\ \sin\beta)$,代入乘加规则:

$$\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}|\,|\mathbf{b}|(\cos\alpha\cos\beta + \sin\alpha\sin\beta) = |\mathbf{a}|\,|\mathbf{b}|\cos(\alpha - \beta)$$

用到的只是三角课里的差角公式。结果只依赖两个长度和方位角之差 α − β——
也就是夹角 θ,和坐标轴画在哪毫无关系。上面"转地面"的做法,不过是挑了
β = 0 那副最省事的坐标。

于是点积有了**同一个量的两副面孔**:

- **代数面孔**便于计算——不需要知道任何角度,乘几下加几下就完了,
  维度再高也是同一套动作;
- **几何面孔**便于理解——它告诉你这一串乘加算出来的数**意味着什么**。

这个"计算走代数、理解走几何"的双面结构,是后面一切的基础。
你拖动演示时对点积的所有直觉,都来自几何那一面。

### 3.4 符号速查表

cos 的尺子传给了点积,于是点积的**符号**天生就带含义:

| 两向量关系 | θ | cos θ | a · b | 一句话 |
| --- | --- | --- | --- | --- |
| 同向 | 0° | 1 | 最大正值 | 完全一致 |
| 锐角 | <90° | 正 | 正 | 大致同向 |
| 垂直 | 90° | 0 | **0** | 互不贡献 |
| 钝角 | >90° | 负 | 负 | 大致相反 |
| 反向 | 180° | −1 | 最小负值 | 完全唱反调 |

> 物理直觉:**功 = 力 · 位移**($W = \mathbf{F} \cdot \mathbf{d} = |\mathbf{F}|\,|\mathbf{d}|\cos\theta$)。推箱子时只有沿运动方向的
> 那部分力在做功;向心力始终垂直于运动方向,所以永远不做功——正是"点积为 0"
> 的情形。

顺带一提两个常用的副产品,后面都会用到:

- **自己点自己 = 长度的平方**:$\mathbf{a} \cdot \mathbf{a} = |\mathbf{a}|^2$(θ=0,cos=1);
- **从点积反解夹角**:$\cos\theta = \dfrac{\mathbf{a} \cdot \mathbf{b}}{|\mathbf{a}|\,|\mathbf{b}|}$——把长度除掉,
  还原出那把纯粹的方向尺子。这个量有个更响亮的名字:**余弦相似度**。

## 四、神经元的几何:一次点积就是一次"验方向"

现在进入正题。前两节的所有直觉,都是为了看懂这一节。

### 4.1 神经元的全部工作

神经网络里的一个神经元,拿到输入向量 x =(x₁, x₂, …, xₙ),
做的事情只有一件:

$$\text{打分} = \mathbf{w} \cdot \mathbf{x} + b \qquad \text{(和自己的权重向量 w 做点积,再加个偏置 b)}$$

如果你玩过本合集的「什么是神经网络」演示(`demos/what-is-nn/`),
那里每个"打分小组"拿 196 个旋钮对 196 个像素做的"乘了权重再求和",
写成一行数学就是 w · x——**和本演示里 a · b 的算法一模一样,
只是维度从 2 变成了 196**。

于是几何解读可以整个平移过来。把 w 看成**神经元关心的方向**,
把 x 看成**当前的输入**:

> **神经元的打分 = 输入 x 在 w 方向上的影子长度 × |w|。
> 它在问:「你这个输入,在我关心的方向上,影子有多长?」**

- x 和 w 方向一致 → 影子长 → 分数高,神经元强烈兴奋;
- x 和 w 垂直 → 影子为零 → 神经元毫无反应(第二节那个"零贡献");
- x 和 w 反向 → 分数为负 → 神经元被抑制。

「什么是神经网络」里说每个神经元"举着一张模板给输入打分,打的是像不像"——
现在你知道这句话的数学本体了:**模板就是 w 这个方向,"像"就是 cos θ 大,
打分就是投影**。

### 4.2 决策边界:一条与 w 垂直的线

神经元打的分有正有负,正负的分界线在哪?在 **w · x + b = 0** 的地方。
在二维里这是一条直线,在高维里是一张"超平面",而 **w 恰好垂直于它**:

```
                ^  w   (神经元关心的方向, 垂直于边界)
                |
                |      w·x + b > 0    兴奋区: x 的影子够长, 打分为正
                |
================+================    决策边界: w·x + b = 0
                |
                       w·x + b < 0    沉默区: 影子不够, 打分为负
```

这张图值得多看一会儿,它是"神经网络如何思考"的最小单元:

- **w 定方向**:边界一定垂直于 w。想让边界转个角度?转 w 就行;
- **b 定位置**:b 把边界沿 w 方向平移。b 越大,边界向"沉默区"退,
  兴奋区越大——它调的是"多像才算像"的门槛;
- **打分即距离**:w·x + b 的大小(除以 |w|)就是 x 到边界的**有向距离**。
  分数越正,x 在兴奋区扎得越深。

> **一个神经元 = 在输入空间里立一张与 w 垂直的隔板,把空间切成两半,
> 然后报告每个输入落在哪一侧、离隔板多远。**

顺带就把 ReLU 也看懂了:ReLU 把负分数压成 0,几何上就是——
**隔板背面的一切,一律当作"影子为零"处理**。神经元从此只对自己那半边空间
"睁眼",成为一个真正有选择性的探测器("没有这个特征"和"有一个反特征"
从此是两回事)。这也是为什么在 what-is-nn 演示里关掉 ReLU 后,
两层网络会塌缩成一层:没有隔板的截断,一串点积连乘还是一个点积,
再多层也只能画一条直的边界。

### 4.3 一层网络:一排隔板同时验方向

一层有 64 个神经元,就是 64 个方向 w⁽¹⁾…w⁽⁶⁴⁾、64 张隔板同时立在输入空间里。
输入 x 进来,得到 64 个影子长度——这 64 个数合起来,就是那一层的输出向量。

> **矩阵乘法 $\mathbf{y} = W\mathbf{x}$ 没有任何新东西:它只是把"对 64 个方向各做一次点积"
> 打包写成了一个式子。矩阵的每一行,就是一个神经元的 w。**

所以"深度网络的前向传播"翻译成几何语言就是:拿输入在第一层的一排方向上
逐一验影子,把影子长度们拼成新向量,再到第二层的一排方向上验影子……
**一层一层地换坐标系、量投影**。GPU 之所以是 AI 的引擎,
只是因为它擅长同时做海量的"乘加"——也就是海量的点积。

## 五、升到高维:embedding 与注意力

维度升到几百上千之后,点积的几何意义不但没有失效,反而成了大模型里
最核心的两件事的原理。

### 5.1 词向量:方向即语义

大语言模型把每个词(token)表示成一个几千维的向量,叫 embedding。
这些向量的**方向编码语义**:意思相近的词,向量方向也相近。

怎么衡量"方向相近"?正是第三节末尾那个余弦相似度:

$$\cos\theta = \frac{\mathbf{a} \cdot \mathbf{b}}{|\mathbf{a}|\,|\mathbf{b}|} \qquad \text{(点积除掉长度,只留方向分)}$$

「国王」和「王后」的向量夹角很小,cos 接近 1;「国王」和「香蕉」接近垂直,
cos 接近 0。搜索引擎的语义检索、推荐系统的"猜你喜欢"、RAG 里的向量库查询,
底层都是这一个动作:**拿两个方向对影子**。

### 5.2 注意力:query 在一排 key 方向上验影子

Transformer 的注意力机制,核心公式的核心就是一堆点积(Q·Kᵀ 的每个元素):
当前位置发出一个 **query 向量 q**("我在找这样的信息"),序列里每个位置
举着一个 **key 向量 k**("我这里有这样的信息")。每对 q·k 做一次点积——

> **和 4.1 的神经元打分是同一个动作:q 在每个 k 的方向上量影子,
> 影子越长,说明"你有的"和"我要的"方向越一致,注意力就越往那里分。**

区别只在于:神经元的 w 是训练后**固定**的模板,而注意力的 q、k 是根据
当前内容**临时算出来**的——相当于每个词每次都现场定制自己的"关心方向"。
但打分那一下,仍然是你在本演示里拖着两个箭头看了半天的 a · b。

## 六、回到演示:你拖动的就是一个神经元

最后把整条线收拢:

1. **cos** 是一把只看方向、不看长短的"一致度尺子":+1 同向,0 垂直,−1 反向;
2. **点积** = 影子 × 长度:cos 打方向分,长度供力度;代数上它只是"乘加",
   所以任何维度都算得又快又并行;
3. **神经元** = 用 w · x + b 验一次方向:w 是它关心的方向,打分是输入的影子长,
   决策边界是与 w 垂直的一张隔板,ReLU 让它只对半边空间睁眼;
4. **一层网络** = 一排隔板同时验方向,矩阵乘法只是点积的打包写法;
5. **embedding 与注意力**的相似度打分,仍然是同一个点积。

所以,回到演示拖一拖:把 a 想成输入,b 想成某个神经元的权重 w。
你把 a 拖近 b 时黄色影子变长、分数上涨——那一刻发生的事,
和一个万亿参数的大模型认出「7」、或者注意力把目光投向某个词时
发生的事,**是同一件事**。

维度从 2 到 4096,点积没有变;变的只是你再也画不出那张图。
而"投影"这个动作还会再回来一次:后续的「最小二乘」演示,做的正是
把数据向量投影到模型空间——到时候你会发现,还是这只影子。
