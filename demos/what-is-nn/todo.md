**旁注：人真的不是这样"看"的吗？** 上面说机器"不像人一样看"，其实只在
表面上成立——**人看图的底层机制，和这套算术惊人地一致，两者并没有本质区别。**
看看你自己的视觉系统在干什么：
- 视网膜上约 1.2 亿个感光细胞，做的事就是把落在自己那格的光强转换成神经
  放电的频率——**字面意义上的"把图像变成一串数字"**，和 196 个像素值
  没有本质区别；
- 下一站的视网膜神经节细胞，对一小片感光细胞的输出做"中心兴奋、周边抑制"
  的**加权求和**——正是后文要讲的点积运算，它的权重图案同样能画成一张
  小模板；
- 再往上，初级视皮层（V1）的"简单细胞"（Hubel 和 Wiesel 因此获得诺贝尔奖
  的发现）是一个个边缘探测器：各自只对**特定位置、特定朝向**的亮暗边最
  兴奋，数学模型就是"线性滤波 + 非线性阈值"——和步骤 2 要讲的"打分小组 =
  加权求和 + ReLU"是同一个公式。

从感知机、神经认知机到卷积网络这条人工神经网络的
血脉，本来就是照着视觉皮层的这些发现设计的。当然，差异仍然存在：大脑有
大量自上而下的反馈连接，有眼动和注意力的主动采样，学习机制也不是反向传播
（这仍是开放问题），而主观"看见"的体验从何而来，至今无人能答。

那么上面这段话真正想戳破的是什么？是"看毫不费力"这个**错觉**。你感觉
"看到一个7"是瞬间的、整体的、自带概念的——但这种毫不费力的体验，本身
就是几百亿个神经元做完海量乘加运算之后交付的**成品**；人类无法感知自己的"像素层"
的内部运作机制，所以误以为看是免费的。这样一来，这个演示展示的就不是一种
与人无关的把戏，而是**把你自己的视觉皮层每天在干、但你永远感觉不到的事，
摊开在 196 个数字的尺度上给你看**。


### 还差一个小开关：ReLU

实际的神经元在打分之后还有一个小动作：**如果分数是负数，就归零**
（术语叫 ReLU，你可以叫它"不及格就闭嘴"规则）。

为什么需要它？因为它让神经元有了"发言/沉默"两种状态：模板匹配上了就报告
分数，没匹配上就干脆闭嘴（输出 0），而不是输出一个含糊的负数去搅浑水。
后面你会看到，正是这个"闭嘴"机制，让 64 个神经元的组合能表达非常复杂的判断。


**一个反直觉的验证。** 我们做过这样的实验：把 196 个像素按同一套固定方式
打乱（图片碎成人眼完全看不懂的雪花），重新训练——成绩几乎不变。这说明网络
学到的"形状"从来就不是几何意义上的，而是"插槽共现集合"；打乱后它照样学出
同样的共现结构，只是那些模板在人眼里成了雪花。**"形状"看起来是几何的，
只因为数据的相关性来自真实世界的几何笔画；网络只是相关性的忠实雕像。**
这也解释了它的局限：单层模板只能记"固定位置集合的共现"，所以形状绑定位置
（见步骤 4 的居中实验）；要得到"任何位置的横线"这种真正几何化的概念，
需要把同一模板复制到所有位置去扫描（这正是后来卷积网络的发明动机）。


**第三层：偏置和 ReLU 是一把刀，切出半个空间。** w·x + b = 0 在 196 维
空间里定义了一张**超平面**（高维里的"平面"），w·x + b 的值正比于输入点
到这张平面的带符号距离；偏置 b 决定平面推离原点多远——"及格线"画在哪
（对应步骤 2 里那个 ReLU 开关）。ReLU 再把负的一侧整个拍成 0：空间被切成两半，只有
"含这个笔画特征"的那半边有输出，离平面越远（特征越明显）输出越大。
64 个小组就是 64 把刀从不同角度切这个 196 维空间，把它分割成大量小房间；
第二层再对这 64 个"带符号距离"加权投票——整个网络其实是在用 64 张
超平面，拼出一个**分段线性的判别面**，把"是 3 的区域"和"是 8 的区域"
在高维空间里围出来。
